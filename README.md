# food-nutrition-analysis-system
# An Intelligent Database System for Food Nutrition Analysis and Prediction

## Overview
This project develops a normalized relational database system for storing and analyzing food nutrition data. It integrates ETL, SQL analytics, and machine learning for calorie prediction.

## Objectives
- Design a 3NF database
- Perform ETL for data cleaning
- Run SQL queries for insights
- Build ML models for calorie prediction
- Visualize results

## Dataset
Indian Food Nutrition Dataset (Kaggle)

## Technologies
- MySQL
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Steps to Run
1. Run SQL scripts from `/sql`
2. Run `etl_load.py`
3. Run `ml_model.py`
4. Run `visualization.py`

## Models Used
- Linear Regression
- Decision Tree
- Random Forest

## Result
Random Forest achieved best accuracy.
